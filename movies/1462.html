KIM BASINGER está excepcional neste suspense arrepiante sobre a luta de uma mulher para salvar a sobrinha de uma terrível maldição. Para Maggie O´Connor (BASINGER), a oportunidade de criar a sobrinha é um sonho ue se realiza. Quando, a menina de seis anos é raptada por uma entidade oculta, o sonho de Maggie se transforma num pesadelo horripilante.
